# 🏛️ نظام مكتب الرفاه المحاسبي - النسخة الإبداعية (Creative Edition V3)

نظام محاسبي متكامل وذكي مصمم خصيصاً لإدارة العمليات المالية والعملاء بكفاءة عالية واحترافية.

## ✨ الميزات الرئيسية

### 💰 إدارة مالية متقدمة
- تسجيل الأرباح والمصروفات بسهولة
- دعم العملات المتعددة (دولار، ليرة تركية، ليرة سورية)
- حساب صافي الربح تلقائياً
- رسوم بيانية يومية للأرباح (آخر 30 يوم)

### 👥 إدارة العملاء
- إضافة وتعديل بيانات العملاء
- تتبع الأرصدة لكل عميل بدقة
- إرسال كشف حساب عبر الواتساب مباشرة
- إدارة المعاملات المالية مع كل عميل

### 📊 تقارير محاسبية شاملة
- ملخص الميزانية حسب العملة
- توزيع الأرباح والمصروفات
- تصدير إلى Excel و PDF
- نسخ احتياطية وعمليات استيراد

### 💱 أدوات متقدمة (جديد!)
- **محول عملات ذكي**: تحويل المبالغ بين العملات بأسعار صرف حقيقية
- **إدارة أسعار الصرف**: تحديث أسعار الصرف الحالية
- **إحصائيات ذكية**: تحليل الديون والعملاء الأكثر نشاطاً

### 📋 سجل النشاطات (جديد!)
- تتبع كامل لجميع العمليات (دخول، خروج، إضافة، تعديل، حذف)
- معرفة من قام بأي عملية ومتى
- متاح فقط للمدير لضمان الأمان

### 🔐 نظام أمان متقدم
- إدارة مستخدمين متكاملة (إنشاء، تعديل، حظر، حذف)
- واجهة دخول آمنة بدون بيانات افتراضية
- صلاحيات مختلفة للمدير والمستخدمين العاديين
- تشفير البيانات محلياً

### 🎨 واجهة مستخدم فائقة الجودة
- تصميم عصري وأنيق (ذهبي وداكن)
- تأثيرات حركية سلسة وجميلة
- دعم الوضع الليلي (Dark Mode)
- تجربة مستخدم احترافية على جميع الأجهزة

## 📋 الصفحات الرئيسية

| الصفحة | الوصف |
|--------|-------|
| **لوحة التحكم** | ملخص الأرباح والمصروفات والإحصائيات |
| **الأرباح والمصروفات** | تسجيل وإدارة العمليات المالية |
| **العملاء** | إدارة بيانات العملاء والأرصدة |
| **التقارير** | تقارير محاسبية شاملة وتصدير |

| **سجل النشاطات** | تتبع جميع العمليات (للمدير فقط) |
| **الإعدادات** | إدارة المستخدمين والبيانات |

## 💾 تخزين البيانات

- جميع البيانات تُحفظ محلياً في متصفحك (Local Storage)
- لا تُرسل البيانات إلى أي خادم خارجي
- يمكنك تصدير واستيراد البيانات في أي وقت
- نسخ احتياطية آمنة وسهلة

## 🛠️ التكنولوجيا المستخدمة

- **HTML5**: هيكل الموقع
- **CSS3**: تصميم متقدم مع تأثيرات حركية
- **JavaScript**: منطق البرنامج والوظائف
- **Chart.js**: رسوم بيانية
- **XLSX**: تصدير إلى Excel
- **jsPDF**: تصدير إلى PDF
- **Font Awesome**: أيقونات احترافية
- **Google Fonts**: خطوط عربية جميلة
